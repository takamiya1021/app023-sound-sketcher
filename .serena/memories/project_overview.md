# プロジェクト概要

## プロジェクト名
**Sound Sketcher (サウンドスケッチャー)** - app023

## 目的
キーボードタップでビートを刻み、簡易タイムラインを書き出せる音楽スケッチツール。
ミュージックビデオや映像編集の音楽プロトタイピングに最適。

## 解決する課題
- ビート・リズムのアイデアを素早くスケッチしたい
- 映像編集時のタイミング合わせに使えるタイムラインが欲しい
- DAWを起動するほどではない簡単なビート作りが面倒
- ミュージックビデオ制作時のリズム感の共有が難しい

## ターゲットユーザー
- 映像クリエイター（ミュージックビデオ、動画編集）
- 音楽プロデューサー・トラックメイカー
- ビートメイカー初心者
- AI音楽生成を活用するクリエイター

## コア機能
1. **キーボード演奏**: キー（ASDFGHJK等）にドラム音を割り当て、リアルタイム再生
2. **録音・再生**: タップしたタイミングを記録し再生
3. **タイムライン表示**: 時間軸に沿って打ち込んだ音を可視化
4. **エクスポート**: JSON/CSV形式でタイムライン書き出し
5. **AI提案**: Gemini APIによるビート展開提案
