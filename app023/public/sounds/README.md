# Sound Files

## License

All sound files in this directory are **public domain (CC0)** and free to use without restriction.

## Files

- `kick.wav` - Bass drum (kick)
- `snare.wav` - Snare drum
- `hihat-closed.wav` - Closed hi-hat
- `hihat-open.wav` - Open hi-hat
- `clap.wav` - Hand clap
- `cymbal.wav` - Crash cymbal
- `rim.wav` - Rim shot
- `tom.wav` - Tom drum

## Technical Specifications

- **Format**: WAV (RIFF)
- **Sample Rate**: 44100 Hz
- **Bit Depth**: 16 bit
- **Channels**: Mono
- **Average File Size**: ~8.7 KB per sample

## Usage

These samples are optimized for low-latency playback with Tone.js in the Sound Sketcher application.
